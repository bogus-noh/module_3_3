# git commit skill 사용법 

## git 사용 순서
- Step 1. 해당 세션에서 작업파일들을 `git add`
- Step 2. add 된 파일등를 `git commit` 진행
    - commit 이름과 커밋 메시지는 `git branch` 이름을 따른다. 
- Step 3. `git push` 진행
- Step 4. `./claude/docs/progress.md` 해당 내용을 최신화 
    - [] 형태의 체크 박스를 사용하여, 직관적으로 표현
